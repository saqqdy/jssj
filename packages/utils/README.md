## Install

```bash
# use npm
$ npm install @jssj/utils --save

# use yarn
$ yarn add @jssj/utils

# use pnpm
$ pnpm install @jssj/utils
```

## Usage

```js
const { nextIndex } = require('@jssj/utils')

nextIndex()
```
